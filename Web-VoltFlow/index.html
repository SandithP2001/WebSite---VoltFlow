MONITOR. ANALYZE. SAVE.

The Intelligent Electricity Management Unit is an AI-driven system designed to help households monitor, predict, and optimize their daily electricity usage. By combining IoT-enabled smart sensors, machine learning forecasting, and generative AI recommendations, the platform transforms raw energy data into personalized, real-time insights for each appliance. Users can track their energy consumption, forecast future usage, and receive actionable suggestions to reduce waste and lower costs. Through its interactive web dashboard, the system provides visual analytics and simulation tools to encourage sustainable behavior. This innovative approach empowers households to make data-driven decisions that contribute to both financial savings and environmental sustainability, paving the way for smarter, greener living.

Domain

Domain

Literature Survey

Existing Home Energy Management Systems (HEMS) mostly provide overall energy usage without device-level insights. Research highlights the need for granular IoT monitoring, AI-based prediction, and personalized feedback. IoT advancements allow real-time monitoring and automation, but many systems lack threshold-based control or renewable integration. Machine learning models like XGBoost and neural networks improve forecasting accuracy, while LLMs (e.g., GPT-Neo) can personalize recommendations for user-specific energy optimization. The literature emphasizes combining these domains to achieve scalable, user-centric energy management.

Research Objectives

Main Objective

Develop a unified IoT and AI-based system that intelligently monitors and optimizes household electricity consumption.

Specific Objectives

  • Implement device-level energy tracking with real-time monitoring.
  • Apply machine learning for forecasting future consumption.
  • Develop threshold-based automation for proactive energy control.
  • Integrate LLM-powered recommendations for personalized guidance.
  • Provide multi-tenant support for fair billing in shared properties.
  • Ensure cloud-based scalability, security, and usability.

Research Gap

Despite progress in IoT and AI, current systems:

  • Fail to integrate device-level monitoring, predictive analytics, and personalized feedback in one platform.
  • Lack threshold-based automation and multi-tenant energy allocation.
  • Offer limited context awareness (e.g., weather, tariffs, occupancy).
  • Rarely incorporate renewable energy sources or cloud scalability.
  • Provide few real-world deployments, focusing instead on simulations.
  • This project addresses all these gaps through a unified, cloud-integrated, and AI-enhanced framework.

Research Problem

How can an integrated IoT-based intelligent energy management system be developed that combines real-time monitoring, predictive analytics, and personalized AI-driven insights to optimize household electricity usage, reduce costs, and foster sustainable behavior?

Methodology

The system integrates:

  • IoT Smart Plugs with current (ACS712) and voltage (ZMPT101B) sensors.
  • AWS Cloud Infrastructure using IoT Core, DynamoDB, S3, and Lambda for data collection, analytics, and automation.
  • Machine Learning Models (XGBoost) trained on device-level data for forecasting.
  • Fine-Tuned GPT-Neo LLM generating personalized, appliance-specific energy-saving recommendations.
  • Web Portal built with React (frontend) and Laravel (backend) for real-time visualization, bill estimation, and control.
  • Security & Accessibility: Encryption, and color-blind-friendly themes.
